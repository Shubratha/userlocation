# userlocation
Store user locations and predict the user's home/work locations

## Local installation
1. Clone the repo
2. Activate virtualenv
    ```source usrloc/bin/activate```
3. Run the server
   ```
   cd usrlocation
   python manage.py runserver   
   ```
 4. Enter url in browser
    http://127.0.0.1:8000/locations/index/
